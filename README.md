## run the project

1. clone this repo

	```copy
	git clone https://github.com/ayoubmehd/tanger-med-tech
	```
1. be sure that
   - your MongoDB server is running
   - you have yarn is installed globally run:
	```
	npm install -g yarn
	```

1. install dependencies

	```copy
	yarn
	```
1. start the development server
   ```copy
   yarn dev
   ```
